MUSIC FOLDER

Place your .mp3 music files in this folder.

Requirements:
- Format: .mp3 files only
- License: Make sure you have rights to use the music (royalty-free, owned by you, or properly licensed)
- Quality: Good quality audio files work best

Recommended sources for royalty-free music:
- Pixabay Music (pixabay.com/music/)
- Bensound (bensound.com)
- Free Music Archive (freemusicarchive.org)
- YouTube Audio Library (studio.youtube.com)

The AI agent will randomly select one of these files for each video it creates.

Example files:
- background_music_1.mp3
- inspirational_track.mp3
- calm_ambient.mp3
- upbeat_motivation.mp3
